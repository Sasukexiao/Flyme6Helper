## 版本支持说明
在主题美化6.10.0之前的版本，适配代码都存在于master分支，之后的适配版本都是单独一个分支，
因为适配得版本多了之后，对于单个版本而言，无效的hook实在太多了。因为代码混淆得原因，hook的类名和方法名经常在变。
